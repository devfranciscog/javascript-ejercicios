# Ejercicio 06 - Entero positivo mas pequeño

## Enunciado

Dado un Array de enteros no ordenados llamado nums, devuelve el entero positivo mas pequeño que falte. 

Prueba 1:
Entrada: nums = [1, 2, 3]
Salida esperada: 4

Prueba 2:
Entrada: nums = [3, 4, -1, 1]
Salida esperada: 2

Prueba 3:
Entrada: nums = [7, 8, 9, 11, 12]
Salida esperada: 1
